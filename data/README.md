# Data directory

This repository does **not** store raw or processed experimental data directly.

All sequencing data, intermediate count tables, and derived summary files are hosted
externally (Imperial OneDrive / institutional storage) due to size and data-management
constraints.

This directory documents the **expected data products**, their **formats**, and
their **relationship to the analysis scripts** in this repository.

---

## Overview of data types

The analyses in this repository operate on four main classes of derived data:

1. Plasmid abundance and fitness measurements (RPM / LFC)
2. Concatemerisation metrics
3. Transcriptional activity metrics
4. Transcription factor binding site (TFBS) counts

Raw sequencing files (FASTQ / BAM) are processed externally to generate the
CSV/TSV files described below.

---

## 1. Plasmid abundance and fitness (LFC)

**Description**  
Per-plasmid log fold change (LFC) during pooled growth assays, derived from
read counts at assay start and end.

**Typical filename(s)**  
- `Plasmid_LFC_Summary_with_Sequences.csv`

**Required columns**
- `Plasmid`
- `Average_LFC_*`
- `LFC_StdDev_*`

**Generated by**
- RPM and LFC calculation scripts (Nanopore alignment → RPM → LFC)

**Used by**
- TRACE (`TRACE/`)
- Random forest models
- Mechanism analysis scripts

---

## 2. Concatemerisation metrics

**Description**  
Quantification of plasmid concatemer formation during propagation, computed
from aligned read lengths relative to reference plasmid size.

**Typical filename(s)**  
- `Concatemer_Percentage_Summary.csv`

**Required columns**
- `plasmid` (or `region`)
- `Average_End_Concatemer_%`
- `Std_End_Concatemer_%`

**Generated by**
- Concatemer analysis scripts (BAM + reference FASTA)

**Used by**
- Mechanism analysis
- Pairwise ranking ablation (concatemer feature)

---

## 3. Transcriptional activity metrics

**Description**  
Quantification of spurious transcription arising from plasmid sequences,
measured as read coverage extending beyond annotated regions.

**Typical filename(s)**  
- `reads_out.tsv`

**Required columns**
- `plasmid`
- `tx_left`
- `tx_right`

**Derived columns**
- `tx_total = tx_left + tx_right`

**Generated by**
- High-throughput transcriptional readout pipeline

**Used by**
- Mechanism analysis
- Pairwise ranking ablation (TX features)

---

## 4. Transcription factor binding site (TFBS) counts

**Description**  
Counts of predicted E. coli transcription factor binding sites per plasmid,
computed using FIMO (MEME suite) at scale.

**Typical filename(s)**  
- `FIMO_presence_matrix.csv`

**Format**
- Rows: plasmids
- Columns: transcription factors
- Values: integer site counts

**Generated by**
- TFBS counting pipeline (external MEME/FIMO execution)

**Used by**
- Mechanism analysis
- Pairwise ranking ablation
- TF class–level analyses

---

## Data access

All data files are available upon request or via institutional storage.
Paths to local data files are provided as command-line arguments or
editable variables within individual scripts.

No analysis in this repository requires modification of source code
to accommodate different data locations.

---

## Reproducibility note

This repository provides:
- All analysis logic
- All plotting code
- All model training and evaluation scripts

Reproducing the figures in the associated manuscript requires supplying
the expected input files documented above.
